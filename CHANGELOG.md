# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-04-15

### Added

- Public package metadata for Composer distribution.
- MIT license and repository-friendly `.gitignore`.
- Magento 2.4-compatible data patch for the `bottom_description` category attribute.
- Admin category form WYSIWYG field integration.
- Frontend block and template example for rendering the bottom description.
- Default `catalog_category_view.xml` layout handle to render the content automatically on category pages.
- Project README with installation, usage, and release guidance.
